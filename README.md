# Participant-Metrics-Python
This project analyses participant survey data stored in a CSV file using Python.

# What it does
Reads CSV data using csv.DictReader
Cleans missing values (NaN)
Calculates the Min, max, average for Metric 1 & 2
Yes/No counts for Metric 3
Maps participant responses

# Tools used
Python
CSV module

# Key Features
Data cleaning (handling missing values)
Basic statistical analysis
Dictionary-based data processing

# Business relevance
This project demonstrates how raw survey data can be transformed into actionable insights for decision-making.
